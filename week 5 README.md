# Text Generation using RNN, LSTM and GRU

## Objective
To build and compare Vanilla RNN, LSTM and GRU models for text generation.

## Tools Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Models Implemented
1. Vanilla RNN
2. LSTM
3. GRU

## Results
- RNN learns sequence patterns but struggles with long-term dependencies.
- LSTM performs better for sequence learning.
- GRU gives performance similar to LSTM with faster training.

## Conclusion
LSTM and GRU are more effective than Vanilla RNN for text generation tasks.
