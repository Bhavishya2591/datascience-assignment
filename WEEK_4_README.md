# CIFAR-10 Image Classification using ANN and CNN

## Objective
Build and compare ANN and CNN models on the CIFAR-10 dataset.

## Results

| Model | Accuracy |
|---------|---------|
| ANN | 43.18% |
| CNN | 72.58% |

## Conclusion
CNN significantly outperformed ANN because it preserves spatial information and extracts image features effectively.
